# 像素贪吃蛇游戏设计文档

**日期:** 2025-03-17  
**项目:** 复古诺基亚风格贪吃蛇游戏  
**技术栈:** HTML5 Canvas + JavaScript

---

## 1. 项目概述

复刻经典诺基亚贪吃蛇游戏，采用完整的复古模拟器风格，包含计分系统、暂停功能和8-bit音效。

## 2. 视觉设计

### 2.1 诺基亚手机模拟器外观

- **外壳:** 灰色手机外壳边框
- **屏幕:** 绿色背光LCD屏幕（#9bbc0f 背景色）
- **显示:** 深色像素显示（#0f380f）
- **按键:** 经典诺基亚按键布局（仅视觉，实际控制用键盘）

### 2.2 游戏元素

- **蛇身:** 深色像素块（8x8像素）
- **食物:** 闪烁的像素点
- **分数显示:** 像素字体，屏幕顶部

## 3. 游戏机制

| 功能 | 实现 |
|------|------|
| 控制 | 方向键 / WASD |
| 暂停 | 空格键 |
| 计分 | 每食物10分，本地存储最高分 |
| 难度 | 每5食物速度+10% |
| 结束 | 撞墙或撞自己 |

## 4. 音效设计

- **吃食物:** 短促的 "哔" 声
- **游戏结束:** 下降的音效
- **技术:** Web Audio API 生成8-bit风格

## 5. 技术架构

```
snake-game/
├── index.html      # 主页面
├── css/
│   └── style.css   # 复古诺基亚样式
├── js/
│   ├── game.js     # 游戏逻辑
│   ├── renderer.js # 渲染引擎
│   └── audio.js    # 音效管理
└── assets/
    └── sounds/     # 音效文件（可选）
```

## 6. 文件结构详解

### 6.1 index.html
- 诺基亚手机外壳结构
- Canvas 游戏屏幕
- 按键视觉元素
- 脚本引用

### 6.2 css/style.css
- 手机外壳样式（圆角、阴影、灰色渐变）
- 屏幕绿色背光效果
- 像素字体（Press Start 2P 或类似）
- 响应式布局

### 6.3 js/game.js
- 游戏状态管理
- 蛇的移动逻辑
- 碰撞检测
- 分数计算
- 本地存储最高分
- 键盘事件监听

### 6.4 js/renderer.js
- Canvas 渲染
- 蛇身绘制
- 食物绘制
- 分数显示
- 游戏结束画面

### 6.5 js/audio.js
- Web Audio API 初始化
- 吃食物音效
- 游戏结束音效
- 音量控制

## 7. 游戏循环

```
初始化游戏
    ↓
游戏循环 (setInterval/requestAnimationFrame)
    ↓
更新蛇位置
    ↓
检测碰撞
    ↓
检查是否吃到食物
    ↓
重新渲染
    ↓
循环 / 结束
```

## 10. 用户交互

| 按键 | 功能 |
|------|------|
| ↑ / W | 向上移动 |
| ↓ / S | 向下移动 |
| ← / A | 向左移动 |
| → / D | 向右移动 |
| 空格 | 暂停/继续 |
| R | 重新开始（游戏结束后）|
