# 诺基亚复古游戏掌机设计文档

**日期:** 2025-03-17  
**项目:** 多游戏掌机（贪吃蛇 + 俄罗斯方块）  
**技术栈:** HTML5 Canvas + JavaScript

---

## 1. 项目概述

将现有的贪吃蛇游戏升级为多游戏掌机，集成经典俄罗斯方块，通过诺基亚风格的下拉菜单切换游戏。

## 5. 俄罗斯方块游戏机制

### 5.1 方块类型（7种）

| 方块 | 形状 | 颜色 |
|------|------|------|
| I | 长条 | 青色 |
| O | 正方形 | 黄色 |
| T | T形 | 紫色 |
| S | S形 | 绿色 |
| Z | Z形 | 红色 |
| J | J形 | 蓝色 |
| L | L形 | 橙色 |

### 5.2 控制方式

| 按键 | 功能 |
|------|------|
| ← → | 左右移动 |
| ↓ | 加速下落 |
| 旋转键 | 旋转方块 |
| 空格 | 暂停 |

### 5.3 游戏规则

- 方块从顶部生成，自动下落
- 底部碰撞后固定
- 完整一行消除，上方下落
- 方块堆到顶部游戏结束
- 消行计分：1行100分，2行300分，3行600分，4行1000分

### 5.4 旋转系统

使用简单的旋转矩阵：
```
顺时针旋转90度：
[x, y] → [-y, x]
```
